# 🐤 つぶやきアプリ - DokoTsubu

シンプルなWebベースのつぶやきアプリです。Java（Servlet）とHTML/CSSで構成されており、ログイン機能・投稿機能・投稿一覧表示などの基本機能を備えています。

## 📌 主な機能

- ユーザー登録 / ログイン
- つぶやきの投稿
- 投稿の一覧表示（新着順）
- 入力バリデーション（未入力・制限文字数など）

## 🛠 使用技術

- Java（Servlet）
- HTML /CSS
- bootstrap
